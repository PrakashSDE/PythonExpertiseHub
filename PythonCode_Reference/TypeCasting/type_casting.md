
# Variable Concept

## Implicit Type Casting / Dynamic Typing
 ```python
   x = 10
   x = "Now I'm a string"
  ```
## Explicit Type Casting

 ```python
   x = 10
   y = str(x)
   z = float(x)
  ```

```python
   x = "hello"
   y = list(x) # ['h', 'e', 'l', 'l', 'o']
  ```

```python
   x = "hello"
   y = tuple(x) # ('h', 'e', 'l', 'l', 'o')
  ```

```python
   x = "hello"
   y = set(x) # {'h', 'e', 'l', 'o'}
  ```

```python
   x = "hello"
   y = dict(x) # {'h': None, 'e': None, 'l': None, 'o': None}
  ```

```python
   x = 10
   y = complex(x)
  ```

```python
   x = 10
   y = bool(x)
  ```

```python
   x = 65
   y = chr(x) # ascii_value 'A'
  ```

```python
   x = 'A'
   y = ord(x) # 65 
  ```
